## Features

- Document-based file structure
- Git-based version control system
- Per-paragraph character count
- Content summary

## Installation

Download the application from the Novelizer download page or from [Releases](https://www.github.com/zuhanit/novelizer/releases).

Currently supported platforms:

- macOS
- Windows
- Linux

Planned platforms:

- iOS
- Android

## Explore

Try Novelizer through interactive examples on the [Novelizer Showcase](https://www.novelizer.app/showcase). You can also browse the UI components via [Storybook](https://www.novelizer.app/storybook).

## Themes

Choose from pre-built themes at [Novelizer Theme](https://www.novelizer.app/themes), or create your own with [Novelizer Theme Builder](https://www.novelizer.app/theme-builder). For more details on theme creation, refer to the Theme Instruction documentation.

## Architecture

- **Frontend:** React, Vite, TypeScript
- **Backend:** Tauri, Rust
