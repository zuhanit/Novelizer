## 기능

- 문서 기반의 파일 구조
- Git 기반의 버전 관리 시스템
- 문단 단위 글자수 체크
- 본문 요약 정보

## 설치

노벨라이저 다운로드 페이지에서 어플리케이션을 다운로드하거나, [Releases](https://www.github.com/zuhanit/novelizer/releases)에서 다운로드 할 수 있습니다.

현재 지원되는 플랫폼은 다음과 같습니다:

- macOS
- Windows
- Linux

지원 예정 플랫폼은 다음과 같습니다:

- iOS
- Android

## 살펴보기

[노벨라이저 쇼케이스 홈페이지](https://www.novelizer.app/showcase)에서 편집 가능한 예시를 통해 노벨라이저를 체험할 수 있습니다. 또는, [Storybook](https://www.novelizer.app/storybook)을 통해 UI 구성을 살펴보실 수도 있습니다.

## 테마

[Novelizer Theme](https://www.novelizer.app/themes)에서 미리 만들어진 테마를 선택하거나, [Novelizer Theme Builder](https://www.novelizer.app/theme-builder)에서 원하는 테마를 직접 만들 수 있습니다. 테마 제작에 대한 더 자세한 내용을 알고 싶다면 Theme Instruction 문서를 확인하세요.

## 아키텍처

- **백엔드:** Tauri, Rust
